## Stack

- HTML5 / CSS3 / Vanilla JS (+ raw WebGL2 fluid sim; vendored Three.js r162 for The Wander & The Constellation)
- Canvas 2D scenes · synthesized Web Audio SFX engine (no audio files)
- Inter + JetBrains Mono (Google Fonts); Font Awesome (demos)
- No build step. No npm. Deployed directly via GitHub Pages.
