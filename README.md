# Saptarshi Nag — Portfolio

Personal portfolio site for Saptarshi Nag, AI/ML Engineer.

Built with pure HTML, CSS, and JavaScript — no build step, no frameworks.

## Links

- **Portfolio:** [saptarshi-nag189.github.io/saptarshinag.github.io](https://saptarshi-nag189.github.io/saptarshinag.github.io/)
- **Experience demos:** [saptarshi-nag189.github.io/saptarshinag.github.io/demos](https://saptarshi-nag189.github.io/saptarshinag.github.io/demos/) — six interactive "choose your experience" concepts (Terminal OS, Mission Control, Neural Graph, Signal Decode, F1 Pit Wall, Movie Premiere)

## Stack

- HTML5 / CSS3 / Vanilla JS (ES Modules)
- Canvas 2D themed backgrounds (no WebGL/Three.js)
- Inter + JetBrains Mono (Google Fonts); Font Awesome (demos)
- No build step. No npm. Deployed directly via GitHub Pages.
